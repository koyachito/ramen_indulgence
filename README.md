# ラーメン免罪符

「今日ラーメンを食べてもよい理由がほしい」という欲望に、時刻・食事回数・運動・仕事や学習・日付から肯定理由をつけるWebアプリです。理由を積み上げ、免罪判決と軽い付帯条件を発行します。

## 主な機能

- 現在時刻、食事回数、8000歩以上歩いたか、仕事・学習、日付による理由生成
- 理由を一行ずつ積み上げる免罪符表示
- 完全免罪、条件付き免罪など4種類の判決
- 日付ごとの特別なジョーク理由
- 免罪符PNGの生成・画像共有とGoogle Maps検索URLの生成
- SQLiteによる判決別・ラーメン種類別の統計
- 閲覧者の現在時間帯ごとのラーメン種類分布
- スマートフォン対応
- スクロール表示、発行演出、カウントアップなどのモーション
- デプロイ監視用 `/health` エンドポイント

ログイン、位置情報取得、ユーザー識別は行いません。判決種別とラーメン種類だけを匿名の集計値として保存し、個別の回答内容は保存しません。

## 技術スタック

- Python / FastAPI / Uvicorn
- Jinja2 / HTML / CSS / JavaScript
- SQLite
- pytest

## ローカル起動

Python 3.11以降を推奨します。

```bash
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements-dev.txt
uvicorn app.main:app --reload
```

ブラウザで <http://127.0.0.1:8000> を開きます。

または、プロジェクト直下で次を実行します。

```bash
bash start.sh
```

リモート開発環境を使っている場合、ブラウザの `localhost` はサーバーと別環境です。
エディタの「Ports / ポート」画面で `8000` を転送し、表示された転送URLを開いてください。

## 理由生成ロジック

現在時刻、食事回数、8000歩以上歩いたか、仕事・学習、日付から決定的に理由文を生成します。外部APIや乱数は使いません。該当しない活動理由は省略しますが、否定的な文章は表示しません。

日付理由は `app/date_reasons.txt` に、2月29日を含む366日分を定義しています。

結果画面は「あなたのラーメン欲は、ここに赦されました。」という免罪符調の文面で締めます。これは栄養・医療上の判断ではなく、入力を振り返って楽しむジョークです。

## 画面

- `/` トップページ
- `/diagnosis` 免罪審議フォーム
- `/result` 免罪符（POST）
- `/stats` 全体の判決統計
- `/about` アプリとプライバシー方針
- `/health` 稼働確認
